message property
================

::: {.section .multi-line-signature}
[String](../string-class) message

::: {.features}
final
:::
:::

Message provided to the user when they use the deprecated feature.

The message should explain how to migrate away from the feature if an
alternative is available, and when the deprecated feature is expected to
be removed.

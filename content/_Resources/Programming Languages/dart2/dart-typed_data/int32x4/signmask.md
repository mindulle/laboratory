signMask property
=================

::: {#getter .section .multi-line-signature}
[int](../../dart-core/int-class) signMask
:::

Extract the top bit from each lane return them in the first 4 bits.
\"x\" lane is bit 0. \"y\" lane is bit 1. \"z\" lane is bit 2. \"w\"
lane is bit 3.

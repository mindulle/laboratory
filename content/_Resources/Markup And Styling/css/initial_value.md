Initial value
=============

The **initial value** of a
[CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) property is its
default value, as listed in its definition table in the specification.
The usage of the initial value depends on whether a property is
inherited or not:

- For [inherited properties](inheritance.md#inherited_properties), the
    initial value is used on the *root element only*, as long as no
    [specified value](specified_value.md) is supplied.
- For [](inheritance.md#non-inherited_properties), the initial value
    is used on *all elements*, as long as no [](specified_value.md) is supplied.

You can explicitly specify the initial value by using the
[`initial`](initial.md) keyword.

**Note:** The initial value should not be confused with the value
specified by the browser\'s style sheet.
